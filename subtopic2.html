The Different Types of Machine Learning

URL
Primary Types of Machine Learning
Site Author
 GeeksForGeeks


Abstract
 The website here analyzes the four different types of machine learning. It goes over the topics of supervised learning, unsupervised learning, semi-supervised learning, and reinforcement learning in great detail. Each type of machine learning is defined and has its process explained. Furthermore, examples for each type of learning are given to further illustrate to the reader how they function. Each type of machine learning is also given a list of advantages and disadvantages, so that it becomes easy to see the areas they shine in and the areas they lack in. Finally, the website includes a long list of applications that utilize a type of machine learning.

 The website here is similar to the previous site, as it goes over three types of machine learning excluding semi-supervised learning.
